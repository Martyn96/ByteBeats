ByteBeats

Formula



Info

Using this script you can create music with formulas.

Variable t stands for time. You can use variable t, bitwise operators and sin(), cos(), tan(), log(), ceil(), floor(), random() to create a music formula!


Examples

melody: (t*5&t>>7)|(t*3&t>>70)|(t*7&t>>100)|(t*9&t>>200)

boss battle: ((t>>10) <<((t>>4)%([8,9,10,11][(t>>11)%4]-(t>>15)%4)))

derp song: (t/1e7*t*t+t)%127|t>>4|t>>5|t%127+(t>>16)|t

noise: t*random()
